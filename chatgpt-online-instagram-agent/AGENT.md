## Workflow diário

1. Escolher um ângulo do dia.
2. Criar copy para feed ou story.
3. Revisar PT-BR e encoding.
4. Criar arte 1080x1440 para carrossel ou 1080x1920 para story.
5. Validar dimensões.
6. Fazer dry-run.
7. Publicar somente se:
   - Conta confirmada.
   - Token válido.
   - Arte validada.
   - Texto sem mojibake.
   - Autorização do workspace/tarefa permitir publicação.
