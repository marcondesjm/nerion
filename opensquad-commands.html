Comandos do Opensquad

Lista consolidada dos comandos documentados no projeto para controlar squads, skills, perfil da empresa e ajustes do sistema.

Fonte: AGENTS.md + README.md Escopo: comandos do Opensquad neste repositório

Atalhos principais

Esses são os comandos que acionam o menu e os fluxos principais.

Comando O que faz
/opensquad Abre o menu principal.
/opensquad menu Abre o menu principal.
/opensquad help Mostra a ajuda do sistema.
/opensquad create <descrição> Inicia o fluxo do Arquiteto para criar um squad.
/opensquad run <nome> Executa um squad pelo Runner.
/opensquad edit <nome> <mudanças> Inicia o fluxo do Arquiteto para editar um squad.
/opensquad list Lista todos os squads na pasta squads/.

Opções de sistema

Comandos para skills, perfil e configuração.

/opensquad skills /opensquad install <nome> /opensquad uninstall <nome> /opensquad delete <nome> /opensquad edit-company /opensquad show-company /opensquad settings /opensquad reset

O menu ainda tem atalhos sem comando direto: Create a new squad, Run an existing squad, My squads e More options.

Atalho do dashboard

O README também documenta um fluxo para abrir o escritório virtual local.

Comando Uso
/opensquad dashboard Gera o dashboard do squad no IDE.
npx serve squads/<nome-do-squad>/dashboard Serve localmente o dashboard gerado.
http://localhost:3000 Endereço sugerido para abrir no navegador.

Observação: os comandos acima são os que aparecem documentados no repositório. Se você quiser uma lista de comandos do editor/CLI do ambiente atual, isso é outra camada e não está descrita neste projeto.

Comandos do ambiente

Estes comandos não aparecem nos arquivos do Opensquad, mas são os atalhos que você citou e que normalmente pertencem ao ambiente/cliente de chat.

Comando Função esperada
/clear Limpa a conversa atual ou o contexto visível.
/compact Compacta o contexto para reduzir tamanho e manter o essencial.
/model Troca o modelo da conversa ativa, quando o cliente suporta esse atalho.
/help Mostra a ajuda do ambiente atual, quando esse atalho existe.

Se você quiser, eu posso colocar esses quatro comandos no topo do HTML e separar em duas abas visuais: Opensquad e Ambiente.

Exemplos rápidos

Exemplo Resultado esperado
/opensquad crie um squad para escrever posts no LinkedIn sobre IA Cria um squad a partir de linguagem natural.
/opensquad execute o squad meu-squad Executa o squad solicitado.
/opensquad crie um squad para [o que você precisa] Mesmo fluxo de criação, com descrição livre.
/opensquad run my-squad Versão em inglês documentada no README.
